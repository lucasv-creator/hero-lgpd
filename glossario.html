GLOSSÁRIO TÉCNICO

Entenda os termos fundamentais da proteção de dados na saúde através deste guia especializado em conformidade legal.

search
LEI L

LGPD

Lei que protege os dados pessoais dos cidadãos no Brasil.

FISCALIZAÇÃO A

ANPD

Órgão do governo responsável por fiscalizar a proteção de dados no Brasil.

AUTORIZAÇÃO C

Consentimento

Permissão dada pela pessoa para que seus dados sejam usados.

PESSOA T

Titular dos Dados

Pessoa dona das informações pessoais.

USO DE DADOS T

Tratamento de Dados

Qualquer uso feito com informações pessoais.

PRIVACIDADE D

Dados Sensíveis

Informações mais íntimas da pessoa, como saúde, religião, biometria e vida sexual.

RISCO V

Vazamento de Dados

Quando informações pessoais ficam expostas ou são acessadas por pessoas não autorizadas.

RISCO V

Violação de Dados

Uso errado, indevido ou ilegal das informações pessoais.

ENVIO C

Compartilhamento de Dados

Quando informações são enviadas para outras pessoas, empresas ou instituições.

DIREITO E

Exclusão de Dados

Pedido para apagar informações pessoais guardadas por uma empresa ou instituição.

DIREITO D

Direito de Acesso

Direito de saber quais dados uma instituição possui sobre você.

CLAREZA T

Transparência

Obrigação de explicar de forma clara como os dados são usados.

CONTROLE F

Fiscalização

Verificação para saber se a lei está sendo cumprida.

RESPONSABILIDADE R

Responsabilização

Obrigação de responder por erros, prejuízos ou uso indevido dos dados.

MOTIVO LEGAL B

Base Legal

Motivo permitido pela lei para usar dados pessoais.

PROTEÇÃO A

Anonimização

Processo que retira informações que identificam uma pessoa.

DECISÃO C

Controlador

Empresa ou instituição que decide como os dados serão usados.

EXECUTOR O

Operador

Pessoa ou empresa que usa dados seguindo orientações de outra instituição.

REGULARIDADE C

Conformidade

Quando uma instituição segue corretamente as leis e regras.

ORGANIZAÇÃO G

Governança de Dados

Organização e controle do uso das informações pessoais.

CANAL DE CONTATO D

DPO / Encarregado

Pessoa responsável por cuidar dos assuntos ligados à proteção de dados na instituição.

SAÚDE P

Prontuário Médico

Documento com informações sobre consultas, exames, diagnósticos e tratamentos do paciente.

SAÚDE D

Diagnóstico

Identificação de uma doença ou condição de saúde.

SAÚDE D

Dados Clínicos

Informações relacionadas à saúde do paciente.

SAÚDE H

Histórico Médico

Registro de doenças, exames, consultas e tratamentos anteriores.

ATENDIMENTO T

Telemedicina

Atendimento médico realizado pela internet.

RECEITA P

Prescrição Médica

Receita ou orientação feita por profissional de saúde.

EXAMES E

Exames Laboratoriais

Testes feitos para avaliar a saúde do paciente.

LOCAL I

Instituição de Saúde

Hospitais, clínicas, laboratórios, postos de saúde e unidades de atendimento.

SEGREDO S

Sigilo Médico

Obrigação de manter as informações do paciente em segredo.

IDENTIFICAÇÃO B

Biometria

Dados físicos usados para identificar uma pessoa, como digital, rosto ou voz.

SEGURANÇA C

Criptografia

Proteção dos dados usando códigos de segurança.

ARMAZENAMENTO B

Banco de Dados

Local digital onde informações ficam guardadas.

TECNOLOGIA S

Servidor

Computador que guarda sistemas, arquivos e informações.

PROTEÇÃO S

Segurança Digital

Proteção contra invasões, golpes, perdas e vazamentos.

VULNERABILIDADE I

Incidente de Segurança

Problema que coloca dados pessoais em risco.

ACESSO A

Autenticação

Confirmação da identidade de uma pessoa ao acessar um sistema.

CÓPIA B

Backup

Cópia de segurança das informações.

AJUDA C

Canal de Denúncia

Local usado para relatar problemas, abusos ou irregularidades.

PEDIDO FORMAL R

Requerimento

Documento usado para fazer um pedido formal.
